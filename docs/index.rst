Welcome to dandelion-eu's documentation!
========================================

Connect to the dandelion.eu_ API in a very pythonic way!

.. note::
    This is a beta package, and may change dramatically. Not all the features are
    implemented as yet, but many are next to come. Please be patient, and perhaps
    write us to let us know you're out there, waiting for something.


Installation
------------

``dandelion-eu`` is available on :samp:`https://pypi.python.org/pypi/dandelion-eu/` install it simply with::

    pip install dandelion-eu
